# Task Management System - Client

## 📋 Mô tả dự án

Task Management System Client là giao diện người dùng được xây dựng bằng Angular, cung cấp một hệ thống quản lý công việc hiện đại và dễ sử dụng. Ứng dụng này giúp người dùng tổ chức, theo dõi và quản lý các nhiệm vụ công việc một cách hiệu quả.

## 🛠 Công nghệ sử dụng

### Core Technologies
- **Angular 19** - Framework frontend
- **TypeScript** - Ngôn ngữ lập trình
- **RxJS** - Reactive programming library
- **Angular Router** - Routing và navigation

### UI/UX Libraries
- **Angular Material** - Material Design components
- **TailwindCSS** - Utility-first CSS framework

### Internationalization
- **@ngx-translate** - Đa ngôn ngữ (Tiếng Anh, Tiếng Việt)

## ✨ Tính năng hiện có

### 🔐 Xác thực và Ủy quyền
- Đăng nhập/Đăng ký tài khoản
- JWT Authentication

### 📊 Dashboard
- Tổng quan dự án và nhiệm vụ
- Biểu đồ thống kê
- Thông tin nhanh về tiến độ công việc

### 📋 Quản lý Nhiệm vụ
- **Danh sách nhiệm vụ**: Xem và quản lý tất cả nhiệm vụ
- **Tạo/Sửa nhiệm vụ**: Form tạo và chỉnh sửa nhiệm vụ
- **Quản lý trạng thái**: Cập nhật trạng thái nhiệm vụ
- **Phân loại nhiệm vụ**: Quản lý loại và ưu tiên
- **Cài đặt dự án**: Cấu hình project settings

### 👤 Quản lý Người dùng
- **Hồ sơ cá nhân**: Xem và chỉnh sửa thông tin cá nhân

## 🚀 Cài đặt và Chạy dự án

### Yêu cầu hệ thống
- **Node.js** >= 16.0.0
- **npm**

### Cài đặt dependencies

```bash
npm install
```

### Chạy development server

```bash
ng serve
```

### Build production

```bash
ng build
```

Build artifacts sẽ được lưu trong thư mục `dist/`.

### Chạy tests

```bash
ng test
```


## 🔧 Scripts có sẵn

- `ng serve` - Chạy development server
- `ng build` - Build ứng dụng
- `ng test` - Chạy unit tests
